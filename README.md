1. Our app allows users to create profiles for their DND characters. On the character select screen, they have the option to create a character, with a name, class, and abilities. You can add as many abilities as you want, or delete them, as well as set the amount of uses they have. The characters you select will be placed in a list. When you select them, you will then be taken to their dedicated page. Here you have the ability to edit their HP, add new abilities, increment abilities, set and view ability info, and take both long and short rests to refresh uses and HP.
